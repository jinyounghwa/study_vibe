# Interactive Docs

An interactive documentation platform built with Next.js and React. This project provides an engaging way to explore and learn from interactive documentation.

## Features

- Interactive content with React components
- Syntax highlighting for code examples
- Smooth animations with Framer Motion
- Responsive design with Tailwind CSS
- Markdown support for flexible content creation
- Chart visualization with Recharts

## Tech Stack

- **Frontend Framework**: Next.js 16
- **React Version**: 19.2.3
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Code Highlighting**: React Syntax Highlighter
- **Markdown Processing**: react-markdown & gray-matter
- **Icons**: Lucide React
- **Charts**: Recharts

## Getting Started

### Prerequisites

- Node.js 18 or higher
- npm or yarn

### Installation

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

### Build

Build for production:

```bash
npm run build
npm start
```

## Project Structure

- `src/app/` - Next.js app directory
- `src/components/` - Reusable React components
- `src/data/` - Data and content files
- `src/lib/` - Utility functions and helpers
- `public/` - Static assets

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Start production server
- `npm run lint` - Run ESLint
